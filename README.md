| INSTALL BOT |
|:-----------------------|
- <p align="left">Clone Source:
```sh
cd $home
git clone https://github.com/soroushwilson/wilson_api
```
- <p align="left">First install:
```sh
cd wilson_api
cd .run
chmod 777 adv
bash adv
bash 1
bash 2
bash 3
```
- <p align="left">install bot:
```sh
cd ..
chmod 777 start.sh
./start.sh install
Bot has been installed! you can run bot with:

RUN BOT WITH CLI MOD:
./start.sh

RUN BOT WITH SELF MOD:
./start.sh self


RUN BOT WITH API MOD:
./start.sh api
```
| ANTI CRASH |
|:-----------------------|
```sh
tmux new-session -s script "bash steady.sh -t"
```
- <p align="left">Now run your bot then enter bot's phone number and join code:
`./start.sh -q`
- <p align="left">And you can start bot with sready script:

| FREE SERVER: |
|:-----------------------|
| `./steady.sh -t` |

| BOUGHT SERVER: |
|:-----------------------|
| `./steady.sh -s` |
